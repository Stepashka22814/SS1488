# SS1488 описание:
Официальное ПО для работы специалистов ОУИС ТС Монетка.

# функционал:
1. Мониторинг прохождения каловых масс в канализацию.
- магазины 
- офисы 
- РЦ


2. Мониторинг сорстояния систем VANtuzz.
- контроль расходов воды
- мониторинг ибепойников системы VANtuzz

3. регистрация обращений и обработка уже созданных обращений с SO

4. Управления туалетными службами (SuperSmiv1488 TUA)
- прогрузка говна в сумочные 
- управление службами VANtuzz
  - gofra < парсер говна в канализацию
  - GWS < служба управления внутренними процессами SS1488
  - seranator < управления пересортицы говна 
  - UTM < унитазный транспортный модуль туалетов
